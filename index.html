สวัสดีครับ! ผมชื่อ ภวัต (Phawat )

ปัจจุบันผมเป็นนักศึกษาชั้นปีที่ 1 คณะ IT ที่กำลังเริ่มต้นเรียนรู้ระบบพื้นฐานของการพัฒนาเว็บไซต์ครับ

นี่คือหน้าเว็บแรกที่ผมสร้างขึ้นมาด้วยตัวเองผ่านโครงสร้าง HTML5 ตื่นเต้นและท้าทายมากครับ!

งานอดิเรกและความสนใจ

  • ศึกษาการเขียนโค้ดและทดลองระบบ Git / GitHub
  • เล่นเกมแนววางแผนและฝึกสมอง
  • ดูภาพยนตร์และฟังเพลงในเวลาว่าง

เป้าหมายในการเรียน

เป้าหมายของผมในการเรียนวิชานี้คือ เข้าใจภาพรวมและโครงสร้างการทำงานของ Front-End Development เพื่อที่ในอนาคตจะสามารถนำไปต่อยอดร่วมกับโครงสร้างระบบ Back-End และพัฒนาโปรเจกต์เว็บแอปพลิเคชันที่สมบูรณ์แบบได้ครับ
